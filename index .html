Pengertian HTML

HTML adalah bahasa markup yang digunakan untuk membuat halaman web dan menampilkan informasi dalam browser. HTML terdiri dari kode-kode tag yang menginstruksikan browser tentang bagaimana menampilkan konten.

Sejarah HTML

  • HTML 2.0: 22 September 1995
  • HTML 3.2: 14 Januari 1996
  • HTML 4.0: 18 Desember 1997
  • ISO HTML: 15 Mei 2000
  • XHTML 1.0: 24 Januari 2000
  • XHTML 1.1: 16 Februari 2007

    • Ciri-Ciri Dokumen HTML

  • Ekstensi file berupa .htm atau .html
  • Non case sensitive
  • Tag pembuka dan penutup (beberapa tag tidak memiliki penutup)
  • Tag bisa saling bersarang

    • Struktur Dasar HTML

    <html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    Konten halaman web
  </body>
</html>
